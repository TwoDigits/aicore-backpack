# Happy Coder

> Mobile and web client for remote access to Claude Code and Codex with end-to-end encryption.

- **URL**: https://app.happy.engineering
- **GitHub**: https://github.com/slopus/happy
- **License**: MIT
- **Category**: Claude Code

## Description

Happy Coder enables developers to monitor and control their AI coding sessions from anywhere. It wraps Claude Code and Codex sessions, allowing seamless switching between desktop and mobile control. When you step away from your desk, you can continue monitoring autonomous AI agents from your phone, approve permissions, and handle errors in real-time.

## Key Features

- **Mobile Access**: Native apps for iOS and Android, plus a web client
- **Real-time Voice Support**: Interact with your coding session using voice
- **Push Notifications**: Get alerts for permission requests or errors
- **Instant Device Switching**: Press any keyboard key to return control to desktop
- **End-to-End Encryption**: All code and session data is encrypted for security
- **Open Source**: No telemetry, fully transparent codebase

## Why It's Cool

Ever wondered what your AI coding agent is doing while you're grabbing coffee? Happy Coder solves the "AI autonomy anxiety" by letting you peek into and control your coding sessions from your phone. The end-to-end encryption means you can safely monitor sensitive codebases without security concerns. It's perfect for developers who use Claude Code in autonomous mode but want to stay informed and in control.

## Getting Started

1. **Download the app**:
   - iOS: App Store
   - Android: Google Play
   - Web: https://app.happy.engineering

2. **Install the CLI**:
   ```bash
   npm install -g happy-coder
   ```

3. **Use it instead of Claude Code**:
   ```bash
   # Instead of 'claude', run:
   happy

   # For Codex:
   happy codex
   ```

4. **Switch to mobile**: When you need to step away, the session continues and you can control it from your phone. Press any key on your keyboard to switch back to desktop mode.
